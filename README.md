# Sistema de Tareas – JavaFX & Spring Boot

<img width="1115" height="755" alt="image" src="https://github.com/user-attachments/assets/9879615f-f90e-4ba8-8649-139fd744e1be" />

Sistema de Tareas es una aplicación de escritorio desarrollada con **JavaFX** para la interfaz gráfica y **Spring Boot** para la lógica de negocio y persistencia de datos.

Permite gestionar tareas de forma sencilla mediante operaciones CRUD (crear, modificar, eliminar y listar), integrando tecnologías modernas de Java en una arquitectura por capas. hola

---

## Funcionalidades

- Agregar nuevas tareas
- Modificar tareas existentes
- Eliminar tareas
- Listar tareas en una tabla interactiva
- Persistencia de datos con MySQL
- Integración de Spring Boot con JavaFX
- Arquitectura limpia y organizada

## Arquitectura

El sistema está organizado por capas:

- **Presentation** → Inicialización JavaFX y arranque de la aplicación
- **Controller** → Manejo de eventos de la interfaz
- **Service** → Lógica de negocio
- **Repository** → Acceso a base de datos con JPA
- **Domain** → Entidades del sistema
- **DB** → Configuración de base de datos

Esta estructura facilita el mantenimiento, escalabilidad y buenas prácticas de desarrollo.
